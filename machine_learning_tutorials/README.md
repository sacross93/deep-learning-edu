# 머신러닝 튜토리얼 시리즈

이 튜토리얼 시리즈는 머신러닝의 핵심 개념과 알고리즘을 체계적으로 학습할 수 있도록 구성되었습니다. 각 주제별로 이론, 실습, 퀴즈가 포함되어 있어 완전한 학습 경험을 제공합니다.

## 📚 튜토리얼 구성

### 1. 기초 개념 및 데이터 처리
- **01. 데이터 마이닝 개요**: 데이터 마이닝의 정의, 데이터 유형, 과제 분류
- **02. 데이터 품질 및 전처리**: 잡음, 이상치, 결측치 처리 방법
- **03. 유사도와 거리**: 다양한 거리 척도와 유사도 측정 방법

### 2. 지도 학습 (Supervised Learning)
- **04. 다변량 회귀**: 연속형 변수 예측을 위한 회귀 분석
- **05. 로지스틱 회귀**: 범주형 변수 예측을 위한 분류 방법
- **06. 의사결정나무**: 해석 가능한 분류 및 회귀 모델
- **07. 규칙 기반 학습**: 명시적 규칙을 통한 분류 방법
- **08. 앙상블 학습**: 여러 모델 조합을 통한 성능 향상 (Random Forest, Boosting)
- **09. 인공신경망**: 복잡한 비선형 패턴 학습을 위한 다층 퍼셉트론
- **10. SVM**: 최적 분류 경계를 찾는 서포트 벡터 머신

### 3. 비지도 학습 (Unsupervised Learning)
- **11. 주성분 분석(PCA)**: 차원 축소를 위한 주성분 분석
- **12. K-평균 클러스터링**: 기본 클러스터링 알고리즘
- **13. 계층적 클러스터링**: 계층 구조를 가진 클러스터링 방법
- **14. DBSCAN 클러스터링**: 밀도 기반 클러스터링과 이상치 탐지
- **15. 그래프 기반 클러스터링**: 네트워크 데이터에서의 커뮤니티 탐지

### 4. 연관 규칙 마이닝 (Association Rule Mining)
- **16. Apriori 알고리즘**: 장바구니 분석을 위한 연관 규칙 발견

## 🎯 학습 목표

각 튜토리얼은 다음과 같은 구성으로 되어 있습니다:

1. **이론 문서** (`*_theory.md`): 
   - 핵심 개념과 수학적 배경 설명
   - 알고리즘의 작동 원리
   - 장단점 및 적용 사례

2. **실습 코드** (`*_practice.py`): 
   - 실제 데이터를 이용한 구현 및 실험
   - 시각화를 통한 결과 분석
   - 파라미터 튜닝 및 성능 비교

3. **퀴즈** (`*_quiz.py`): 
   - 학습 내용 확인을 위한 대화형 퀴즈
   - 개념 이해도 평가
   - 상세한 해설 제공

## 🚀 시작하기

### 필수 요구사항
- **Python 3.8 이상**
- **필수 패키지**: 아래 명령어로 설치
  ```bash
  pip install -r requirements.txt
  ```

### 주요 의존성 패키지
- `pandas`, `numpy`: 데이터 처리
- `scikit-learn`: 머신러닝 알고리즘
- `matplotlib`, `seaborn`: 데이터 시각화
- `networkx`: 그래프 분석
- `scipy`: 과학 계산

### 학습 순서
1. **순차적 학습 권장**: 01번부터 16번까지 순서대로 학습
2. **각 주제별 학습 과정**:
   - 이론 문서 읽기 → 실습 코드 실행 → 퀴즈 풀기
3. **실습 환경**: Jupyter Notebook 또는 Python IDE 사용 권장

### 실행 예시
```bash
# 이론 학습 (마크다운 뷰어로 읽기)
cat 01_data_mining_overview_theory.md

# 실습 실행
python 01_data_mining_overview_practice.py

# 퀴즈 풀기
python 01_data_mining_overview_quiz.py

# 통합 테스트 (모든 파일 검증)
python test_all_tutorials.py
```

## 📊 사용 데이터셋

실습에서 사용되는 주요 데이터셋:

### 내장 데이터셋 (scikit-learn)
- **Iris**: 붓꽃 분류 (다중 클래스 분류)
- **Boston Housing**: 주택 가격 예측 (회귀)
- **Wine**: 와인 품질 분류
- **Breast Cancer**: 유방암 진단 (이진 분류)
- **Digits**: 손글씨 숫자 인식 (이미지 분류)

### 생성 데이터셋
- **합성 클러스터링 데이터**: 다양한 형태의 클러스터 구조
- **소셜 네트워크 시뮬레이션**: 그래프 클러스터링용
- **장바구니 데이터**: 연관 규칙 마이닝용

## 🛠️ 유틸리티 모듈

`utils/` 디렉토리의 공통 유틸리티:

- **`data_utils.py`**: 
  - 데이터 로딩, 분할, 전처리 함수
  - 샘플 데이터셋 생성 함수
  
- **`evaluation_utils.py`**: 
  - 분류, 회귀, 클러스터링 성능 평가
  - 교차 검증 및 모델 비교
  
- **`visualization_utils.py`**: 
  - 데이터 분포 시각화
  - 모델 결과 시각화
  - 성능 비교 차트
  
- **`quiz_utils.py`**: 
  - 대화형 퀴즈 시스템
  - 점수 계산 및 피드백

### 자주 발생하는 문제
1. **패키지 설치 오류**: 
   ```bash
   pip install --upgrade pip
   pip install -r requirements.txt
   ```

2. **한글 폰트 문제**: 
   - matplotlib 한글 폰트 설정 확인
   - 시스템에 한글 폰트 설치

3. **메모리 부족**: 
   - 대용량 데이터 실습 시 배치 크기 조정
   - 불필요한 변수 삭제 (`del` 사용)
